Próximos Passos

Roadmap para Produção

Quatro horizontes de evolução — sem reescrever um nó sequer do workflow atual.

Imediato · 1 semana
Troca de modelo LLM
llama3.1:8b → gemma2:2b — latência cai de 15 s para ~7 s sem custo algum.
Curto prazo · 1 mês
GPU + cache diário
GPU NVIDIA (Ollama CUDA) leva latência a < 2 s. Cache em Sheets elimina chamadas repetidas.
Médio prazo · 3 meses
WhatsApp Business API
Migração do sandbox Twilio para API oficial — escala ilimitada e consentimento LGPD formal.
Longo prazo · 6 meses
Multi-campus & painel admin
Expansão para múltiplos campi, multi-idioma e interface web para gestores configurarem o pipeline.
